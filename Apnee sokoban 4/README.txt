Comment utiliser le code:

make pour compiler
make clean pour supprimer les .class

Pour lancer le Sokoban avec une grille en particulier, donner son nom en argument, la grille doit
etre dans res/Grille pour etre lue, le format doit etre le bon

Quand le programme est lance, la touche S permet a l'utilisateur de rechercher une seance de coups resolvant le niveau affiche
La touche T quand appuye permet de jouer un pas de la sequence, maintenir pour que le deplacement soit continu
